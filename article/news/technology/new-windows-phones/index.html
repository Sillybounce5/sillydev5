New Windows Phones?

By Silly 12-07-2022



Since the windows phone flopped due to horrible software, Microsoft seems to have been pushing more and more updates to make windows better to use on touch screen devices, and leakers speculate that these updates are leading towards a more mobile future for Microsoft. Other leaks appear to show recent mobile patents made by Microsoft for a touchscreen phone with an under screen camera, a technology which succeeded on the Samsung Z Fold 3.



Some people think that a "Windows Mobile OS" would just flop again, and that Microsoft may make android powered phones, or make a phone OS which isn't affiliated with windows.
But one thing that we know for sure is that whenever Microsoft makes a phone, it will be 99% bloatware
