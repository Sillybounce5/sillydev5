Getting Started

Requirements

Note: while you can create Minecraft: BE addons on Linux, MacOS, and iOS devices, it is reccommended to create them on Windows or Android

  • Minecraft: BE (For iOS, Android or Windows)
  • A code editor (VS Code for Windows, Mac & Linux, Koder for iOS, DroidEdit for Android)
  • Image editor that supports very low-res images (paint.net for Windows, 8-Bit Painter for Android & iOS, GIMP for Mac & Linux)

Setting Up

Using a file explorer create a folder with the name of the addon you are planning to create, and inside that folder create 2 folders, {Addon Name}_Res & {Addon Name}_Beh.


Inside the Res & Behavior folders create a 'manifest.json' file, JSON files are what minecraft uses for behaviours, loot tables, 3D Models and more.


Now go to This website change the type to resource and behaviour (Dependent) and fill in the required information


Copy the left box into the manifest.json file in the RES folder, and vice versa for the BEH folder

Using your image editor create a square image that will appear as the pack icon, and save it in both folders as pack_icon.png
