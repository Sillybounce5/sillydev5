Custom Items

Make item behaviour

To make minecraft items you will need to go into the BEH folder that you created in step 1 and create a folder named items inside this folder you will need to create a file called {Item Name}.item.json. Open the file and paste this template in:

{ "format_version": "1.10", "minecraft:item": { "description": { "identifier": "my:item", }, "components": { "minecraft:hand_equipped": false, "minecraft:stacked_by_data": true, "minecraft:foil": false, "minecraft:max_stack_size": 64, "minecraft:max_damage": 10 } } }

Inside the components section are the necessary components needed for items, but more components can be added to customize your item such as the ones listed here

The "identifier" variable is what the game uses to identify the item (This is not the items display name) for example, dirt uses the identifier "minecraft:dirt". But you cannot use "minecraft:" at the start of the identifier, you have to use your own starter, such as "sillydev5:", and try to keep it the same on all items, blocks, and mobs

Make item resource

Open your chosen photo editor and create a 16x16 texture for your item and save it to RES/textures/items/{Item Name}.png (Make sure it has a transparent background)

Create a file called item_texture.json in RES/textures and paste in:

{ "resource_pack_name": "Cool Pack", "texture_name": "atlas.items", "texture_data": { "{your item name here}": { "textures": "textures/items/{your item name here}" } } }

This file defines shortcuts to the item textures

Now create a new file called {Item Name}.json at RES/items and paste in:

{ "format_version": "1.10", "minecraft:item": { "description": { "identifier": "my:item", "category": "Nature" }, "components": { "minecraft:icon": "{your item name here}", "minecraft:render_offsets": "tools" } } }
