Texture Packs

What is a texture pack?

A texture pack is an addon that edits the textures that already exist in the game. In Minecraft you can edit the textures for: Mobs, Items, Blocks, and you can even edit the game's shaders.

Texture Packs vs. Resource Packs

Texture packs and resource packs in Minecraft are 2 different things. Texture packs are a type of resource pack, and resource packs include texture packs, and textures for custom items

How to make a texture pack

To make a texture pack you need to download the games official resource pack, and an image editor (Some are listed here. Minecraft currently only allows certain texture resolutions, for blocks & items these resolutions are 16x16, 32x32, 64x64, and 128x128, for mobs the textures can have any aspect ratio but the highest resolution for any side is 256.
